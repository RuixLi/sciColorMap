# Discussions

> The record of exploratory work - converging by discussion, one thread per question or issue. Delete these `>` notes. Open a thread, **explore -> narrow -> record** the decision, then close it; the closed thread is its own record, so exploratory work needs no separate log (`how-to-work-here`). Follows `writing-for-the-user`.
